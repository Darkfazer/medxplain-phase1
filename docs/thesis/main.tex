%% ── Abstract ──────────────────────────────────────────────
\begin{abstract}
\noindent
Medical imaging volumes are growing faster than expert radiologist capacity,
and existing artificial-intelligence tools rarely account for the stringent
privacy, interpretability, and workflow requirements of real hospital
environments. This thesis presents \medxplain, an end-to-end
\emph{Medical Visual Question Answering} (VQA) framework designed for
direct hospital deployment under HIPAA and GDPR constraints.

The system ingests raw \dicom{} studies from hospital PACS systems,
performs deterministic pseudonymisation of Protected Health Information
(\textsc{PHI}), and converts pixel data into normalised tensors compatible
with modern vision encoders. A pluggable two-encoder architecture pairs a
BiomedCLIP vision backbone with a BioGPT language decoder, fused via a
trainable cross-attention module. Training proceeds in two phases: Phase~1
aligns image and text embedding spaces using NT-Xent contrastive loss
(frozen backbones), while Phase~2 fine-tunes the full model for VQA using
Low-Rank Adaptation (LoRA, rank~=~8) to minimise GPU memory requirements.

Every model prediction is accompanied by a Grad-CAM heatmap overlay
surfaced to the clinician in a Gradio web application that never transmits
data outside the hospital network. A clinical validation module computes
Cohen's Kappa agreement against radiologist ground-truth and generates
PHI-free discrepancy PDF reports for expert review. An enhanced evaluation
suite reports BERTScore~F1 and per-pathology Clinical~F1 alongside
standard exact-match accuracy and BLEU-4, targeting the benchmarks set in
the project Cahier des Charges ($>$65\%, $>$0.25, $>$0.75, $>$0.60
respectively).

Experimental results are presented on a de-identified chest radiograph
cohort, and ablation studies quantify the benefit of the two-phase
training strategy and LoRA rank selection. All code is open-source,
offline-capable, and fully documented.

\bigskip
\noindent
\textbf{Keywords:} Medical VQA, DICOM, HIPAA, BiomedCLIP, BioGPT, LoRA,
Grad-CAM, Explainable AI, Radiology, Deep Learning.
\end{abstract}
